# Legal Document Comparator

A privacy-first, local legal document comparison tool that uses AI to identify differences in clauses between two legal documents.

## Features

- **Privacy-First**: All processing happens locally on your machine. No data ever leaves your computer.
- **Smart Comparison**: Uses embeddings to find similar clauses and LLM verification to confirm semantic matches.
- **Multi-Format Support**: Handles PDF and DOCX documents.
- **Interactive UI**: Built with Streamlit for easy document upload and results visualization.
- **Detailed Reports**: Generate comprehensive reports showing clause-level differences.
